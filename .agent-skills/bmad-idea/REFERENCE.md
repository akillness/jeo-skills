# bmad-idea Reference

`bmad-idea` is now the repository's **pre-planning creative router**.
Its source of truth is `SKILL.md`; this file exists as a quick compatibility reference for older BMAD-CIS naming.

## Modern source-of-truth modes

| Modern mode | When to choose it | Primary artifact |
|-------------|-------------------|------------------|
| Problem framing | The main gap is still the problem definition | `problem brief` |
| Audience and value framing | The main gap is user / JTBD / positioning / differentiation clarity | `positioning brief` |
| Concept shaping | The idea needs one tighter concept with constraints and scope boundaries | `concept brief` |
| Game concept framing | The packet is a pre-production game concept | `game concept packet` |
| Story packaging | The idea exists but needs a clearer narrative or pitch | `story / pitch packet` |
