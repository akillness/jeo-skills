# Triage
- Problem: Users need a current, source-backed way to install Vercel-style agent skills and understand which global or project directory each operating system and agent should use.
- Audience: Maintainers and users of `oh-my-skills` who install skills into Claude Code, Codex, Gemini CLI, OpenCode, or the portable `.agents/skills/` layer.
- Why now: Vercel released the open `skills` CLI and skills.sh ecosystem in 2026, and this repo needed to add locally installed Claude `deepinit` and `deep-dive` skills to its installable surfaces.

Provenance: direct page retrieval from Vercel Agent Skills docs, Vercel changelog, Vercel KB, and `vercel-labs/skills` README.
