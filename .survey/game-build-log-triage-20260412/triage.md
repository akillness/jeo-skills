# Triage
- Problem: The skill catalog has broad developer coverage but no strong reusable skill for Unity/Unreal build and editor log triage, even though game developers repeatedly hit long, noisy failure logs and mixed-tool debugging loops.
- Audience: Solo developers, indie teams, gameplay programmers, technical designers, technical artists, and build/release owners working in Unity or Unreal Engine projects.
- Why now: Recent .agent-skills additions focused on general AI/dev tooling, while game development remains under-covered. Cross-engine log triage is a bounded, transferable workflow that can deliver immediate value without risky code mutation.
