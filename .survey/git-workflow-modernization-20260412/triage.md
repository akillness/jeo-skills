# Triage
- Problem: `git-workflow` is a legacy catch-all Git cheat sheet that blurs local Git mechanics, PR collaboration, and recovery guidance instead of acting like a modern routing skill.
- Audience: AI coding agents and human developers who need safe branch/commit/rebase/push/recovery workflows in active repositories.
- Why now: The repo has been successfully modernizing older high-utility anchors with sharper trigger boundaries, support files, and evals; `git-workflow` is still broad, outdated, and highly reusable across CLI/developer-workflow tasks.
